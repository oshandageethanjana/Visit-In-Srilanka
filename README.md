# Serendib Yatra

A premium Sri Lanka travel blog and destination storytelling platform built with **PHP**, **MySQL**, **Tailwind CSS**, and a modern dark UI.

Serendib Yatra showcases famous places in Sri Lanka through rich blog content, featured destination stories, legal pages, and an admin panel for managing posts, categories, authors, and tags.

visit live - https://srilanka.whitecoder.online/

---

## Preview

- Homepage: `index.html`
- Blog listing page: `blogs.php`
- Single blog page: `post.php`
- Admin dashboard: `admin.php`
- Privacy Policy: `privacy-policy.html`
- Terms of Service: `terms-of-service.html`

---

## Features

### Frontend
- Premium Sri Lanka travel landing page
- Blog listing page with:
  - featured hero slider
  - latest story cards
  - author and category display
- Single post page with:
  - featured image hero
  - author info
  - category label
  - view count
  - rich HTML blog content
- Privacy Policy page
- Terms of Service page
- Responsive design
- Dark modern UI with green accent theme
- Tailwind CSS based styling
- Preloader animations
- Mobile navigation menu

### Admin Panel
- Secure admin login
- Dashboard statistics
- Create, edit, delete blog posts
- Toggle featured posts
- Duplicate posts
- Manage authors
- Manage categories
- Manage tags
- Search and filter posts
- SEO fields for posts:
  - meta title
  - meta description
  - slug

### SEO / Technical
- Clean URL support using `.htaccess`
- `robots.txt`
- `sitemap.php` / `sitemap.xml`
- Canonical URL support
- Open Graph meta tags
- Twitter card tags
- JSON-LD schema support for posts
- 404 page
- favicon / logo support

---

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache / XAMPP recommended
